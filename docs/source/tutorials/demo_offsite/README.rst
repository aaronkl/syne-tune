Demo for Syne Offsite March 2023
================================

This tutorial provides a walk-through of Syne Tune, mostly from a user's
perspective. You will learn how to use Syne Tune in order to rapidly compare
methods on tabulated benchmarks, and how to add your own benchmark for
evaluations.
